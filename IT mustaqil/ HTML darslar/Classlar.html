IP SAN refers to a storage network architecture that utilizes Internet Protocol (IP) to transfer and manage data between servers and storage devices. It relies on storage protocols that are based on the standard Internet Protocol suite established by the Internet Engineering Taskforce

he most common type of IP SAN uses iSCSI (Internet Small Computer Systems Interface) to encapsulate SCSI commands and assemble data into packets for efficient communication between the host servers and the storage devices.
